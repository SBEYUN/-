## О проекте

Шифратор Цезаря — веб-приложение и консольная программа для шифрования и дешифрования текста методом циклического сдвига (шифр Цезаря).

### Возможности

- Поддержка русского (с буквой Ё) и английского алфавитов
- Сохранение регистра (заглавные/строчные буквы)
- Небуквенные символы (цифры, пробелы, знаки) остаются неизменными
- Автоопределение режима — программа сама решает, шифровать или расшифровывать
- Копирование результата в буфер обмена
- Очистка поля ввода (крестик)
- Адаптивный веб-интерфейс в бежево-белой гамме

## Запуск веб-версии

1. Скачайте все файлы в одну папку
2. Откройте файл `index.html` в любом браузере
3. Файл `fon.jpg` должен лежать в той же папке

Никаких дополнительных установок не требуется.
