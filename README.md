## О проекте

Шифратор Цезаря — это веб-приложение и консольная программа для шифрования и дешифрования текста методом циклического сдвига (шифр Цезаря). Проект разработан в рамках учебной дисциплины и демонстрирует базовые принципы криптографии.

### Возможности

- Поддержка **русского** (с буквой **Ё**) и **английского** алфавитов  
- Сохранение **регистра** (заглавные/строчные буквы)  
- Небуквенные символы (цифры, пробелы, знаки препинания) остаются **неизменными**  
- **Автоопределение режима** — программа сама решает, нужно шифровать или расшифровывать текст  
- **Копирование результата** в буфер обмена  
- **Очистка поля ввода** (крестик)  
- Адаптивный веб-интерфейс в бежево-белой гамме с фоновым изображением Юлия Цезаря  

## Запуск веб-версии

1. Скачайте все файлы в одну папку  
2. Откройте файл `index.html` в любом браузере  
3. Фоновое изображение `fon.jpg` должно лежать в той же папке  

> **Никаких дополнительных установок не требуется** — всё работает в браузере.
